## **About This Project**
Sistem ini adalah **web sederhana untuk manajemen inventory**, dibuat menggunakan **Laravel**.  
Fitur utama:
- Login & Logout
- CRUD Post (sebagai item/inventory)
- Status item (Tersedia / Habis)
- Search & Pagination di daftar post
- Rapi dan mudah dikembangkan

> Cocok sebagai tugas atau contoh sistem inventory sederhana dengan Laravel.

## **Features**
| Feature | Description |
|---------|-------------|
| Login & Logout | Hanya pengguna terdaftar yang bisa mengakses dashboard |
| CRUD Posts | Tambah, lihat, edit, hapus post/item |
| Status Badge | Menandai item Tersedia / Habis |
| Search & Pagination | Mempermudah pencarian dan navigasi daftar item |
| Responsive | Bisa diakses di desktop dan mobile |
